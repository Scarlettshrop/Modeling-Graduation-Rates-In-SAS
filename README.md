# Modeling-Graduation-Rates-In-SAS

## Purpose 
Analyze the influence of demographic and quantitative features on IPEDS-reported graduation rates using SAS. Performed extensive feature engineering to process model selection procedures for improved predictive analysis. Developed Linear and Logistic Regression models to analyze the strength of predictor effects on graduation rates.


The goal of this project was to apply a linear regression to predict graduation rates and logistic regression to predict if a graduation rate is abuoev the median using IPEDS data. For both of these, we limited the incoming cohort to being greater than 200 students. Specification sheets, model selection process, and final models are included for both linear and logistic regression.
